## Tarefa 005 - 10/06/2022 - Linguagens de Programação
1. **Java** - É uma das linguagens mais utilizadas no mundo, possui diversas aplicações, tem tipada forte e  orientação a objetos.
2. **TypeScript** - suporta o paradigma de Programação Orientada a Objetos e pode ser tipado de forma optional sendo esses o diferencial dessa linguagem com o javascript.
3. **pyton** - é uma linguagem de programação de alto nível(mais proximo a linguagem humana) interpretada de script, imperativa, orientada a objetos, funcional, de tipagem dinâmica e forte, seu maior foco é na legibilidade do código.
4. **C** - É uma linguem estruturado e tipada que comparada com liguagens mais atuais é de mais baixo nivel, ou seja, mais proximo a linguagem de maquina. Uma vantagem da linguagem C é que pode ser usada para varias finalidades podendo criar sistemas operacionais, aplicativos de todos os tipos, drivers e outros controladores de dispositivos,etc. Sendo uma linguagem que deu origem a varias outras como c#,c++.
5. **Php** - Hypertext Preprocessor é uma linguagem de programação não tipada e o ponto principal que o diferencia das outras linguagens é a possibilidade de escrever o codigo junto com o html
Ex:
~~~
<h1>
<?
 echo "Teste";
?>
</h1>
~~~

## Linguagem especificada
- **Java** - A sintaxe do java foi derivada do C++ e sua principal caracteristica é todo o codigo é escrito dentro de uma classe e tudo é um objeto tirando os tipos primitivos de dados(boleano, inteiro, caracteres). A linguagem Java foi criada com o objetivo de ser mais um modelo mais simples que a linguagem C e possui uma arquitetura neutra e portável, para poder ser utilizada em diversos Sistemas operacionais, ter alta performance, apresentar segurança e solidez e ser uma linguagem interpretada, a linguagem Java pode ser executada em qualquer plataforma que possua a Java Virtual Machine (JVM) instalada, independente da arquitetura do computador.
